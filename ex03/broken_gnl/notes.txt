
I
#include <unistd.h> for read
#include <stdlib.h> for malloc and free

II
helper functions:

1. string length calculator (ft_strlen)
- return 0 if input is NULL

2. string joiner with free (ft_strjoin_free)
- returns new joined string but frees one prior constituent

3. newline finder
- returns index where needle was found in stack

4. freeing helper
- accepts 2 strings, frees them if not == NULL

III
variable setup:

- static char *stash: holds data from buffer that has not been printed yet
- char *buffer: temporary array to hold bytes from read with BUFFER_SIZE + 1
- ssize_t bytes: to store returned value of read
- char *line: what is to be returned
- char * temp: placeholder
- int newline_i: index of newline
- int i: for incrementing through line

IV
pre-work
- malloc buffer; safeguard should free stash and set it to NULL
- newline_i = find_newline(stash); (will be -1 on first run)

V
loop
- condition: while newline_i == -1
- call read() with buffer as parameter and byte counter as target
- if byte counter == -1: free stash and set to NULL,return NULL for Error
- if counter == 0: done reading, break loop
- null terminate the buffer
MERGING
- if stash still == NULL
	- ft_strjoin_free into stash with only buffer, usual safeguard
- else:
	- ft_strjoin_free stash and buffer into temp (frees stash), usual safeguard
	- set stash to temp
- recall find_newline on newline_i

VI
extract the line
- if newline_i == -1 --> means that read reached EOF, loop broke
	- free buffer
	- temp = stash
	- stash = NULL
	- return (temp)
- malloc line for newline_i + 1 + 1 (for newline and null term) + safeguard
- while i <= newline_i (smaller OR EQUAL!)
	- line[i] == stash[i]; i++;
- top off with null term on i
- set temp = NULL
- if there is still something in the stash after the newline (stash[newline_i + i] != '\0')
	- strjoin_free stash + newline_i + 1 into temp + safeguard

VII
clean up
- free(stash);
- stash = temp (if there was something after newline, it is now stored there for future calls; otherwise NULL)
- free(buffer)
- return (line)
