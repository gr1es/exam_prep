# GET_NEXT_LINE BLUEPRINT

I. INCLUDES
- <unistd.h> (read)
- <stdlib.h> (malloc, free)
#ifndef BUFFER_SIZE 42

II. HELPER FUNCTIONS
1. ft_strlen
   - Return 0 if input is NULL.
2. ft_strjoin_free(char *s1_to_free, char *s2_to_keep)
   - Mallocs size: len(s1) + len(s2) + 1.
   - If s1 is NULL: Copies only s2.
   - If s1 exists: Copies s1, then s2, then frees s1.
   - NEVER frees s2.
3. find_newline
   - Returns index of '\n' or -1 if not found.
4. free_all
   - Checks if pointers are not NULL before freeing.

III. VARIABLES
- static char *stash; (Persists between calls)
- char *buffer;       (malloc size: BUFFER_SIZE + 1)
- ssize_t bytes;      (Stores read() return)
- char *line;         (The returned string)
- char *temp;         (For swapping pointers)
- int newline_i;      (Index of newline)

IV. PRE-WORK
- Malloc buffer.
  - Safeguard: if fail -> free(stash), stash=NULL, return NULL.
- newline_i = find_newline(stash).

V. READING LOOP
- Condition: while (newline_i == -1)
  1. bytes = read(fd, buffer, BUFFER_SIZE).
  2. Error Check (bytes == -1):
     - free_all(stash, buffer), stash = NULL, return NULL.
  3. EOF Check (bytes == 0):
     - break loop.
  4. Null terminate: buffer[bytes] = '\0'.
  5. Merge to Stash:
     - If (!stash): stash = ft_strjoin_free(NULL, buffer).
     - Else:        stash = ft_strjoin_free(stash, buffer).
     - Safeguard both calls!
  6. newline_i = find_newline(stash).

VI. EXTRACT THE LINE
- EOF Handling (if newline_i == -1):
  - If stash has data: return stash (cleanup buffer, set stash=NULL).
  - If stash empty: cleanup all, return NULL.
- Malloc line (size: newline_i + 2).
  - Safeguard.
- Copy loop: from 0 to newline_i (inclusive) into line.
- Null terminate line.
- Save Remainder (temp):
  - temp = NULL.
  - If (stash[newline_i + 1] != '\0'):
    - temp = ft_strjoin_free(NULL, stash + newline_i + 1).
    - Note: Pass NULL as 1st arg to avoid invalid free.
    - Safeguard.

VII. CLEANUP
- free(stash).
- stash = temp (updates static var to the remainder).
- free(buffer).
- return (line).
