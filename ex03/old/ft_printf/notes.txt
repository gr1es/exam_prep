
1. variable number of function parameters
	- function declaration:
		int	ft_printf(const char *format, ...)
	- #include <stdarg.h>
	- va_list arg_list
		- stores parameters
	--> va_start(arg_list, format);
		- format = last parameter before variable range of parameter
	- every future call of va_arg(va_list X, variable type Y) delivers the next parameter
		that is stored in the va_list variable X
		- Y has to correspond to that parameter's variable type
	- va_end(arg_list).

3. general stuff:
	- reset g_bytes to 0 at the start of ft_printf!
	- empty string (printf("");) just prints nothing and returns 0; NOT -1!
	- int to str: str needs buffer of 12 bytes!
		--> INT_MAX/INT_MIN has 11 digits (including null term)
	- when differentiating between non-% and %dir:
		- when skipping directing %, return -1 if following char is null terminator
		--> no lone %, org printf returns -1 if dir is missing
	- format == NULL and %s == NULL are differnt scenarios
		- format == NULL returns -1 (Error)
		- string == NULL prints (null) and returns 6
	- str[12] vs malloc
		- use malloc if string with designated memory should be returned
		- str[12] is fine if usage of str is done before return()
	- remember malloc protection!
	- hardcode 0 for hex conversion! 0 in dec --> 0 in hex
	- handle INT_MIN (-2147483648) for int_to_str!
